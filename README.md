## How to run?

### Configuring .env file
```
PORT = 5000
MONGOURI = YOUR MONGOURI
```

### Installing dependencies
```
npm run client-install
npm install
```

### Running the server
```
npm run dev
```

### Running the test
```
npm run test
```



### N.B: Must have Node js installed in your machine and npm to run the App
